Propiedad transition - otros apuntes

(Posicionar el cursor sobre la bicicleta)

transition: ;

Permite que el paso del estado inical al estado final de un elemento o propiedad del mismo, no se efectúe de manera brusca.

Ejemplo 1:

(Así quedaría si NO se aplica la propiedad transition)
.figura1:hover{ transform: translate(900px);}

transition-property: ;

Con este valor se indican las propiedades que van a ser animadas: all | none | propiedad CSS.

transition-duration: ;

Para que una transición funcione,hay que especificar la duración. Unidades: s | ms

Ejemplo 2:

.figura2{ transition-property: transform;
transition-duration: 6s; }

.figura2:hover{ transform: translate(900px); }

transition: ;

Ejemplo 3:

.figura3{ transition-property: border-radius, transform;
transition-duration: 5s; }

.figura3:hover{ border-radius: 50%;
transform: translate(900px); }

transition: ;

Ejemplo 4:

.figura4{ transition-property: all;
transition-duration: 5s; }

.figura4:hover{ border-radius: 50%;
transform: translate(900px);
background-color: yellow; }

transition-timing-function: ;

Permite que el usuario vea la animación como progresiva a velocidad constante o con cierta aceleración. Sus valores son: ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier.

transition-timing-function: ease;

Inicio lento, transcurso rápido, final lento.

Ejemplo 5:

.figura5{ transition-property: all;
transition-duration: 3s;
transition-timing-function: ease; }

.figura5:hover{ transform: translate(900px); }

transition-timing-function: linear;

Inicio normal, transcurso normal, final normal. Movimiento uniforme.

Ejemplo 6:

.figura6{ transition-property: all;
transition-duration: 3s;
transition-timing-function: linear; }

.figura6:hover{ transform: translate(900px); }

transition-timing-function: ease-in;

Inicio lento, transcurso normal, final normal.

Ejemplo 7:

.figura7{ transition-property: all;
transition-duration: 3s;
transition-timing-function: ease-in; }

.figura7:hover{ transform: translate(900px); }

transition-timing-function: ease-out;

Inicio normal, transcurso normal, final lento.

Ejemplo 8:

.figura8{ transition-property: all;
transition-duration: 3s;
transition-timing-function: ease-out; }

.figura8:hover{ transform: translate(900px); }

transition-timing-function: ease-in-out;

Inicio lento, transcurso normal, final lento.

Ejemplo 9:

.figura9{ transition-property: all;
transition-duration: 3s;
transition-timing-function: ease-in-out; }

.figura9:hover{ transform: translate(900px); }

transition-timing-function: cubic-bezier(A, B, C, D);

Es una función personalizada, en donde A: punto de partida, B: posición que debe ocupar dentro del total una vez transcurrido el tiempo, C: tiempo final de la animación, D: posición final.

Ejemplo 10:

.figura10{ transition-property: all;
transition-duration: 3s;
transition: cubic-bezier(0.25, 0.1, 0,25, 1); }

.figura10:hover{ transform: translate(900px); }

transition-delay: ;

Define cuánto tiempo tardará en que comience la transición. (El tiempo de espera para que ésta se ejecute.). Unidades: s | ms

Ejemplo 11:

.figura11{ transition-property: all;
transition-duration: 3s;
transition-timing-function: linear;
transition-delay: 2s; }

.figura11:hover{ transform: translate(900px); }
